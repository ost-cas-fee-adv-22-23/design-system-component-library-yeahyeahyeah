## Font units

| size | px   | rem      | lineHeight | _label sizes_ |
| ---- | ---- | -------- | ---------- | ------------- |
| xxs  | 12px | 0.75rem  | 0.75rem    |               |
| xs   | 14px | 0.875rem | 0.875rem   | **small**     |
| sm   | 16px | 1rem     | 0.875rem   | **medium**    |
| base | 18px | 1.125rem | 1.575rem   |               |
| md   | 20px | 1.25rem  | 1.575rem   | **large**     |
| lg   | 24px | 1.5rem   | 1.575rem   | **xlarge**    |
