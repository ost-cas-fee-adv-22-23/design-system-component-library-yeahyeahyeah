# Font

## Font Poppins, sans-serif

### Reference

Google Font\*\* [Poppins](https://fonts.google.com/specimen/Poppins 'Google Fonts: Poppins')

## Default Font Size:

| Property    | Value           |
| ----------- | --------------- |
| font-size   | 18px / 1.125rem |
| line-height | 25px            |
| color       | _slate 900_     |
